# Desafio Devs do RN

## Instalação

1. Clone o repositório:
   ```sh
   git clone https://github.com/sergioh665/DesafioPHP
   cd DesafioPHP
   ```
2. Importe o banco de dados:
   ```sh
   mysql -u root -p < meu_database.sql
   ```
3. Inicie o servidor embutido do PHP:
   ```sh
   php -S localhost:8000 -t public
   ```
